# AI Trend Analysis

## 1. Problem Statement

The objective of this project is to analyze AI and Data Science job market trends using statistical and visualization techniques. The analysis focuses on salary patterns, experience levels, company characteristics, remote work trends, and organizational adoption patterns to generate meaningful business insights.

---

## 2. Dataset Description

Dataset Name: AI Job Market Dataset

Total Records: 15,000

Features Used:

- job_id
- job_title
- salary_usd
- salary_currency
- experience_level
- employment_type
- company_location
- company_size
- employee_residence
- remote_ratio
- required_skills
- education_required
- years_experience
- industry
- posting_date
- application_deadline
- job_description_length
- benefits_score
- company_name

The dataset contains job postings from different industries and organizations across multiple locations.

---

## 3. Trend Analysis

The following trends were analyzed:

- Industry vs Salary Trend
- Company Size vs Salary Distribution
- Experience Level vs Benefits Score
- Remote Work Ratio Analysis
- Job Description Length Trends
- Industry-wise Hiring Patterns

Visualizations Used:

- Line Charts
- Bar Charts
- Scatter Plots
- Heatmaps

---

## 4. Statistical Findings

### Descriptive Statistics

Calculated:

- Mean
- Median
- Standard Deviation
- Minimum
- Maximum
- Quartiles

### Correlation Analysis

Studied relationships between:

- Salary and Experience
- Benefits Score and Salary
- Remote Ratio and Benefits Score
- Company Size and Experience

### Hypothesis Testing

ANOVA Test was performed to determine whether company size significantly affects salary distribution.

Null Hypothesis (H₀):
Company size has no significant impact on salary.

Alternative Hypothesis (H₁):
Company size significantly impacts salary.

---

## 5. Visual Insights

Key observations:

- Higher experience levels generally receive higher salaries.
- Large companies tend to offer better compensation packages.
- Benefits score shows a positive relationship with salary.
- Remote opportunities vary across industries.
- Some industries demonstrate higher demand for skilled professionals.
